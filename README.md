# Tutorial Inicial do Django
## Instalação
~~~
pip install django
~~~

## Criando um projeto
~~~
django-admin startproject nomediretorio
~~~

## Para saber qual a versão do Django pelo terminal
~~~
$python3
>> import django
>> print(django.get_version())
~~~

## Iniciando uma aplicação
~~~
python manage.py startapp nomeaplicacao
~~~

## Para rodar a aplicação
~~~
python manage.py runserver
~~~
