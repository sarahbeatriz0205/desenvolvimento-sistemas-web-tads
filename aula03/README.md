# Django - Ciclo Requisição - Resposta
- **Interface Gateway:** Serve para fazer a comunicação com o Python
- **Request Middleware:** Recebe as informações da requisição e traduz a requisição (até então em alto nível padrão HTTP) para um objeto Python
- **Resolução URL:** Um arquivo ```urls.py``` que se comunica com as Views, mapeando a URL da requisição
- **View:** Métodos Python que processam requisições
